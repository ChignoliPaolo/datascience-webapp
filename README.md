# Piaggio Ticket Search

Text search webapp built with Streamlit framework.

## Getting Started

### Dependencies

**Prerequisites**

* Streamlit
* NLTK
* GoogleTranslate API
* PyEnchant
* Lingua

### Installing

* Download files from this repository.

### Executing program

* In the directory of the project run this Streamlit command:
```
streamlit run src/main.py
```
